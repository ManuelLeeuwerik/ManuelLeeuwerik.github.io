6 Open Peer Review

Auteur: Manuel Leeuwerik

For this open peer review I scored a nature article called: ‘Adipose tissue retains an epigenetic memory of obesity after weight loss’ by Hinte et al. (2024).

This study examines how adipose tissue retains an epigenetic memory of obesity, even after significant weight loss. It explores the molecular basis of this “obesogenic memory” and its impact on metabolic function, including predisposition to8. weight regain. The research combined single-nucleus RNA sequencing (snRNA-seq) of human subcutaneous and omental adipose tissues from individuals undergoing bariatric surgery with mouse models subjected to high-fat diets and subsequent weight loss. Epigenetic analyses in mice included ATAC-seq and CUT&Tag to measure chromatin accessibility and histone modifications. Key markers and pathways were assessed, and functional assays measured glucose and lipid uptake in adipocytes.

The study revealed that both human and mouse adipose tissues exhibit persistent transcriptional alterations following significant weight loss. Notably, obesity-induced changes in the epigenome of mouse adipocytes were found to adversely affect their functionality and responsiveness to metabolic signals. Mice with this “obesogenic memory” experienced accelerated weight regain, and the presence of this epigenetic memory was linked to future transcriptional dysregulation in adipocytes when exposed to a high-fat diet. Overall, these findings underscore the existence of a stable obesogenic memory in mouse adipocytes, likely extending to other cell types, which predisposes them to adverse responses in an obesogenic environment, thereby contributing to the common “yo-yo” effect associated with dieting.

I scored this article on ‘Ripeta’ criteria, see the transparency criteria table below. Visit: https://www.researchgate.net/publication/340244621_Reproducibility_and_reporting_practices_in_COVID-19_preprint_manuscripts for more information regarding the scoring.

I scored another project on the R-code. The project is called:The Role of Risk Preferences in Responses to Messaging about COVID-19 Vaccine Take-up.

Specifically I looked at the file: Study1.Rmd, which was used to to reproduce the results reported in the paper. In terms of readability of the code it gets a 2 (1 being very bad and 5 being very good). There are headers in the Rmd. but no comments, no figure references and no README files.

I first installed the packages and the data file Study1.csv. Next i ran part of the code, from “##COVID-19 standard vaccine” (because the entire code is to much info for this review).

I encountered one error because of an old package called “library(semTable)”. I deleted this package from the setup and ran the (working) code again. I did not take much effort to reproduce the visualization from the project so I score the reproducibility a 4 (1 being very hard and 5 being very easy). For readability in this review i hid the code and made the headers bold text from the original code. Note: there is not much text in the original code file.
